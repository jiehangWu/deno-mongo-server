# deno-mongo-server
This is a example API service built with Deno and MongoDB.
## Endpoints
### Show All Messages
Retrieve a list of all the messages.<br>
**URL:**
/messages <br>
**Method:**
`GET`<br>
### Show Message by Id
Retrieve a single message by its Id.<br>
**URL:**
/messages/:id <br>
**Method:**
`GET`<br>
### Create a new message
**URL:**
/messages <br>
**Method:**
`POST`<br>
